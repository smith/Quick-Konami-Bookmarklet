Quick Konami

What Quick Konami is

These days, it seems that a common trend on the interwebz is to hide easter eggs within a website using the ever famous Konami Code. Quick Konami is a bookmarklet for Firefox that makes life a little easier by automatically keying the Konami Code with the press of a button.

How it works

It couldn’t be easier! Simply drag the Quick Konami button below to your bookmark bar. Next, visit any website using the Konami code and wait for the page to fully load. Finally, click the Konami Code bookmarklet.

Quick Konami

I’ve installed it and it doesn’t work

There are several reasons why it may not work: The most obvious reason is that the website you’re trying it on may not be using the Konami code. For a list of known sites using the Konami code, we recommend visiting konamicodesites.com. Some sites require you to press ENTER after entering the code, in which case you can use the bookmarklet and press ENTER afterwards. Another reason that it may not be using is that you’re not using the Firefox web browser. At this time, Quick Konami is only compatible with Firefox.
